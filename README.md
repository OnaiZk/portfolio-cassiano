# Cassiano Silva - Professional Portfolio

[![React](https://img.shields.io/badge/React-19.0.0-blue.svg?logo=react&logoColor=white)](https://reactjs.org/)

This repository contains the source code for the professional portfolio of Cassiano Silva. It serves as a comprehensive showcase of technical expertise, professional experience, and completed projects, designed with a focus on modern web standards, performance, and user experience.

## Overview

The application is built as a single-page application (SPA) providing a seamless, interactive experience. It features dynamic content loading, theme management, and responsive layouts tailored for all device sizes. 

## Technical Architecture

The project is built upon a modern, high-performance technology stack:

* **Frontend Framework**: React 19
* **Build Tool**: Vite
* **Styling**: Custom CSS with CSS Variables for theming
* **Icons**: Lucide React
* **State Management**: React Hooks (useState, useEffect)

## Key Features

* **Theme Management**: System-aware and user-toggled Light/Dark mode.
* **Performance Optimization**: Implemented with Vite for rapid module replacement and optimized production builds.
* **Interactive UI/UX**: Includes custom cursors, scroll-reveal animations, and a seamless preloader.
* **Component-Based Architecture**: Highly modular structure for maintainability and scalability.
* **Secure Infrastructure**: Follows enterprise best practices, ensuring sensitive configurations and environment variables are securely managed in the deployment environment.

## Project Structure

The codebase is organized following industry-standard practices:

```text
src/
├── components/       # Reusable React components
│   ├── About.jsx     # Biography and background
│   ├── Contact.jsx   # Contact form and links
│   ├── CV.jsx        # Resume viewer
│   ├── Experience.jsx# Professional history
│   ├── Hero.jsx      # Landing section
│   ├── Navbar.jsx    # Main navigation
│   ├── Projects.jsx  # Portfolio items
│   └── Skills.jsx    # Technical proficiencies
├── App.jsx           # Main application entry point
├── main.jsx          # React DOM rendering
└── index.css         # Global styles and theme variables
```

## Security & Deployment

This project adheres to strict security standards suitable for enterprise environments. All environment configurations, API integrations, and deployment details are securely managed on the hosting provider and are intentionally excluded from this repository.
